# frontend--update-api

В api.json описано текущее api сервиса.
Его надо прочитать и обновить текущие типы и ручки для фронтенда.
Также надо добавить queries и хуки для get запросов и mutations и хуки для остальных (изменяющих запросов).

Перед изменениями составить отчет о планируемых изменениях.

DEV ручки не добавляем, refresh-token тоже.
Если обновляются коды в API, для них надо прописать переводы.
