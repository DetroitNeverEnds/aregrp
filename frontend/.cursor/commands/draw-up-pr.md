# draw-up-pr

Оформи ПР по текущей ветки по следующим правилам:

- Все на английском

- Заголовок ПРа формируется, как `<fix|feat|refactor> frontend: <максимальное краткое/обобщающее описание>

- Описание ПРа должно быть по следующим правилам:
```
Added pages:
- <Список страниц>

Edited pages:
- <Список страниц>

Added components:
- <Список общих компонентов (из src/components)>

Edited components:
- <Список общих компонентов (из src/components)>

Other:
- <Список прочих изменений>

```

- Сформированные заголовок и описание надо выставить в github
