## Компоненты, которые нужно добавить

### 1. **Card / Container**
**Описание**: Универсальный контейнер для форм и контента с тенью и скруглением
**Расположение**: `components/ui/common/Card`
**Пропсы**:
- `padding?: 'sm' | 'md' | 'lg'` - размер внутренних отступов
- `shadow?: boolean` - наличие тени
- `borderRadius?: 'sm' | 'md' | 'lg'` - радиус скругления углов
- `children: ReactNode` - содержимое

**Применение**: Используется для обертки форм входа/регистрации и других карточек контента

---

### 2. **Form**
**Описание**: Обертка для форм с автоматической обработкой submit
**Расположение**: `components/ui/common/Form`
**Пропсы**:
- `onSubmit: (data: FormData) => void` - обработчик отправки
- `children: ReactNode` - поля формы
- `className?: string` - дополнительные стили

**Применение**: Стандартизация работы с формами

---

### 3. **FormField**
**Описание**: Обертка для полей формы с автоматическим отображением ошибок
**Расположение**: `components/ui/common/FormField`
**Пропсы**:
- `label?: string` - метка поля
- `error?: string` - сообщение об ошибке
- `required?: boolean` - обязательное поле
- `children: ReactNode` - инпут или другой элемент

**Применение**: Унификация отображения полей форм с метками и ошибками

---

### 4. **Divider**
**Описание**: Разделитель контента (горизонтальная или вертикальная линия)
**Расположение**: `components/ui/common/Divider`
**Пропсы**:
- `orientation?: 'horizontal' | 'vertical'` - ориентация
- `spacing?: 'sm' | 'md' | 'lg'` - отступы вокруг
- `text?: string` - текст в центре разделителя (например, "или")

**Применение**: Разделение секций, например между формой и социальными кнопками

---

### 5. **Alert / Message**
**Описание**: Компонент для отображения уведомлений и сообщений
**Расположение**: `components/ui/common/Alert`
**Пропсы**:
- `variant: 'info' | 'success' | 'warning' | 'error'` - тип сообщения
- `title?: string` - заголовок
- `message: string` - текст сообщения
- `closable?: boolean` - возможность закрыть
- `onClose?: () => void` - обработчик закрытия

**Применение**: Отображение ошибок входа, успешной регистрации и т.д.

---

### 6. **Spinner / Loader**
**Описание**: Индикатор загрузки
**Расположение**: `components/ui/common/Spinner`
**Пропсы**:
- `size?: 'sm' | 'md' | 'lg'` - размер
- `color?: string` - цвет
- `fullscreen?: boolean` - на весь экран

**Применение**: Отображение процесса загрузки при отправке формы

---

### 7. **BackLink**
**Описание**: Специализированная ссылка для возврата назад
**Расположение**: `components/ui/common/BackLink`
**Пропсы**:
- `to: string` - путь назад
- `children: ReactNode` - текст ссылки

**Применение**: Навигация назад на предыдущую страницу (уже реализовано через Link с иконкой)

---

## Приоритет реализации

### Высокий приоритет:
1. **Alert / Message** - для отображения ошибок и уведомлений
2. **Spinner / Loader** - для индикации загрузки
3. **Card / Container** - для унификации контейнеров

### Средний приоритет:
4. **FormField** - для упрощения работы с формами
5. **Divider** - для разделения контента

### Низкий приоритет:
6. **Form** - можно использовать нативный `<form>`
7. **BackLink** - уже реализовано через существующий Link
